 This is the data flow:  index.html ---->FLASK (port 5050) ------> SQL Database

Please follow these instructions:
1. use sql script create-phone-book.sql to create a table. Note the DB(schema) and table name in the file.
2. open app.py and edit the DB config like host IP address, user, password (likes 14 to 17)
3. install flask related packages on PC where flask will run, you can run it on ubuntu also.
	pip install flask
	pip install flask_cors
4 open a terminal window or command prompt and run app.py
